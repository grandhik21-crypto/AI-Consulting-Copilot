# Wikipedia Chatbot

A beginner-friendly chatbot that answers questions using Wikipedia. Built with **Python**, **FastAPI**, and a simple web frontend.

## Project structure

```
wikipedia-chatbot/
├── README.md                 # Project overview and setup instructions
├── backend/                  # Python API server
│   ├── requirements.txt      # Python dependencies
│   ├── app/                  # FastAPI application layer
│   ├── services/             # Business logic (Wikipedia API, chat flow)
│   ├── models/               # Data shapes (request/response schemas)
│   └── tests/                # Automated tests
└── frontend/                 # Web UI served to the browser
    ├── static/               # CSS, JavaScript, images
    └── templates/            # HTML pages
```
